THE 5TH DIMENSION

The 5th dimension is a game that uses the kaboom.js library. The 5th Dimension is a shooter game. You play as the main character: a pilot that works for a secret company that explores and investigates the universe. CONTROLS below.

FEATURES:

  • 5 available leves.
  • A story mode.
  • Different types of enemies.
  • A boss fight

CONTROLS:

  • Up arrow => Move Up
  • Down arrow => Move Down
  • Space => Shoot
  • Enter => Go to the next screen
